# Product Guidelines

## Prose Style
- **Clarity:** Use simple, direct language.
- **Conciseness:** Avoid unnecessary words.
- **Tone:** Professional, informative, and helpful.

## Branding
- **Name:** "EC2-ALB-FastAPI-Reference".
- **Color Palette:** Professional blue and white theme for documentation.

## UX Principles
- **Simplicity:** Minimize steps for deployment.
- **Responsiveness:** Ensure API endpoints return consistent, structured responses.
- **Error Handling:** Provide clear, actionable error messages.
