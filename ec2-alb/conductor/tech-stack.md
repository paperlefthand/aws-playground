# Tech Stack

## Programming Languages
- **Python:** Used for the FastAPI application.
- **TypeScript:** Used for the AWS CDK infrastructure.

## Frameworks
- **FastAPI:** High-performance web framework for Python.
- **AWS CDK:** Software development framework for defining cloud infrastructure.

## Infrastructure
- **AWS EC2:** Virtual servers in the cloud.
- **Application Load Balancer (ALB):** Distributes incoming application traffic.

## Tools
- **Node.js & npm:** For managing AWS CDK dependencies.
- **pip:** For managing Python dependencies.
